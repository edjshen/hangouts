# Hangouts

Event coordination and hangout planning platform.

## Overview

A platform for organizing casual get-togethers, tracking who's coming, and coordinating details.

## Tech Stack

- Backend: Node.js + Express
- Database: PostgreSQL + Prisma
- Frontend: React

## Development

```bash
npm install
npm run dev
```

## Structure

```
src/
  server/       # Express API
  client/       # React frontend
  db/           # Database schema & migrations
tests/          # Test suites
```
